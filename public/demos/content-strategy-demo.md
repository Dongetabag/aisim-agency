## Content Style Guide

### Voice & Tone

| Attribute | Description | Example |
|-----------|-------------|---------|
| **Confident** | We know our stuff | "Here's what actually works..." |
| **Clear** | No jargon | "Simply put..." |
| **Conversational** | Like talking to a colleague | "Let me explain..." |
| **Results-focused** | Everything ties to outcomes | "This led to X result..." |

### Writing Guidelines

**Do:**
- Start with the reader's problem
- Use concrete numbers and examples
- Include actionable takeaways
- Write at 8th-grade reading level
- Use short paragraphs (3-4 sentences max)

**Don't:**
- Use marketing buzzwords
- Overpromise or hype
- Write generic content
- Skip the practical application
- Forget the CTA

### Formatting Standards

- **H2:** Main sections
- **H3:** Subsections
- **Bullets:** Lists of 3+ items
- **Bold:** Key terms, emphasis
- **Callouts:** Important notes, tips
- **Images:** 1 per 300-400 words

## SEO Optimization Checklist

For each article:

- [ ] Primary keyword in title (H1)
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] 3-5 secondary keywords naturally included
- [ ] Internal links to 3+ other articles
- [ ] External links to 2+ authoritative sources
- [ ] Meta description (150-160 characters)
- [ ] Alt text for all images
- [ ] Table of contents for 2000+ word articles
- [ ] FAQ section with schema markup
- [ ] CTA at end of article
